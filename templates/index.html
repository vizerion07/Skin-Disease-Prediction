ABOUT THIS WEBSITE

This website leverages the power of Deep Learning using Convolutional Neural Networks (CNNs) to classify a given image into one of 8 skin diseases and 1 class for Healthy Skin. This project demonstrates how CNNs can be implemented effectively for medical imaging tasks.

These skin diseases include:

  • Cellulitis
  • Impetigo
  • Athlete-Foot
  • Nail-Fungus
  • Ringworm
  • Cutaneous-larva-migrans
  • Chickenpox
  • Shingles
  • Healthy
